# Corvo Mail — Documentação

O **Corvo Mail** é uma interface web para criação e consulta de caixas de e-mail temporárias. O projeto usa a API pública do [mail.tm](https://docs.mail.tm/) para criar contas, autenticar e consultar mensagens.

Site oficial: [corvomail.online](https://corvomail.online)

## Recursos

- Criação automática de caixa descartável
- Cópia rápida do endereço
- Atualização periódica da caixa de entrada
- Leitura e exclusão de mensagens
- Chave de acesso para recuperar uma caixa existente
- Interface responsiva, sem cadastro no Corvo Mail

## Comece por aqui

1. Acesse `corvomail.online`.
2. Aguarde a criação do endereço.
3. Copie o endereço e use-o no serviço desejado.
4. Mantenha a página aberta para acompanhar novas mensagens.
5. Salve o endereço e a chave se quiser tentar recuperar a caixa depois.

## Avisos importantes

O Corvo Mail não opera os servidores de recebimento de e-mail. Os domínios, caixas e mensagens são fornecidos pelo mail.tm e podem expirar ou ficar indisponíveis sem aviso. Não use caixas temporárias para contas importantes, documentos confidenciais, recuperação de senha ou qualquer conteúdo que precise ser preservado.
