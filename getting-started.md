# Perguntas frequentes

## Por quanto tempo a caixa existe?

Não há prazo garantido. A disponibilidade é determinada pelo mail.tm.

## Preciso fazer cadastro?

Não. A caixa é criada automaticamente. A chave exibida é uma credencial da caixa, não uma conta no Corvo Mail.

## Posso usar um endereço @corvomail.online?

Não. Esse é o domínio do site. Os domínios de e-mail são disponibilizados pela API mail.tm.

## Como volto a uma caixa?

Salve o endereço e a chave de acesso. Informe os dois no formulário de recuperação enquanto a caixa ainda existir.

## Perdi a chave. É possível recuperar?

Não. O Corvo Mail não mantém um mecanismo alternativo de recuperação.

## Uma mensagem desapareceu. É possível restaurar?

Não depois que ela tiver sido excluída ou expirada na infraestrutura mail.tm.

## É anônimo?

O serviço reduz a necessidade de informar seu e-mail pessoal, mas não promete anonimato absoluto.

## Alguns sites não aceitam o endereço. Por quê?

Serviços podem bloquear domínios conhecidos de e-mail descartável. O Corvo Mail não consegue contornar essas políticas.

## É indicado para dados importantes?

Não. Use apenas em situações descartáveis e de baixo risco.
