# Privacidade e segurança

## Dados no navegador

O Corvo Mail salva localmente no navegador os dados necessários para manter a caixa ativa entre recarregamentos. Isso inclui endereço, chave de acesso, identificador e token da sessão. Esses dados ficam acessíveis a quem usar o mesmo perfil do navegador.

## Serviço externo

Criação de contas, autenticação, recebimento e armazenamento das mensagens são realizados pelo mail.tm. Consulte as políticas desse serviço antes de usar a aplicação.

## O que não prometemos

O Corvo Mail não promete anonimato absoluto, entrega garantida, criptografia ponta a ponta, retenção definida ou recuperação de conteúdo expirado.

## Boas práticas

- Não use para bancos, trabalho, saúde ou contas importantes.
- Não receba documentos ou informações confidenciais.
- Não compartilhe a chave de acesso.
- Exclua mensagens quando não forem mais necessárias.
- Considere limpar os dados do site em dispositivos compartilhados.

## Conteúdo HTML

Mensagens HTML são abertas em um `iframe` com sandbox para reduzir riscos ao restante da aplicação. Ainda assim, trate links e anexos de remetentes desconhecidos com cuidado.
